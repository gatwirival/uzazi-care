# 📎 File Upload Quick Reference

## ✅ Supported Files

| Type | Formats | Max Size | AI Capability |
|------|---------|----------|---------------|
| **Images** | JPG, PNG, WebP, GIF | 20 MB | Vision analysis, OCR, diagnosis support |
| **PDFs** | PDF documents | 20 MB | Text extraction, summarization |
| **CSV** | Comma-separated | 10 MB | Trend analysis, pattern detection (100 rows) |
| **Text** | TXT, MD, JSON | 1 MB | Content analysis (5000 chars) |

---

## 🚀 3-Step Usage

### 1️⃣ Upload
Click **📎** button → Select file(s)

### 2️⃣ Ask
Type your question about the file

### 3️⃣ Analyze
Get AI-powered insights in real-time

---

## ⚡ Quick Tips

✅ **DO**:
- Use clear, well-lit images
- Include CSV headers
- Ask specific questions
- Upload 1-3 files at a time

❌ **DON'T**:
- Upload blurry/dark images
- Send huge files (compress first)
- Include sensitive identifiers
- Expect instant diagnosis

## 🔧 Troubleshooting

| Issue | Solution |
|-------|----------|
| Slow upload | Reduce file size (compress images) |
| No analysis | Check file format is supported |
| Incomplete data | CSV limited to 100 rows (upload to Files for full analysis) |
| Error message | Check browser console (F12) for details |

---

## ⚠️ Important

- Critical decisions require human expertise
- De-identify patient data before upload
